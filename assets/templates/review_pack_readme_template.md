# <课程名> 期末复习资料包

本资料包由课程 PDF/PPT 课件结构化整理生成，适用于期末短期速成复习。

## 推荐阅读顺序

1. 先读 `00_课程总览.md`，建立课程整体框架。
2. 按优先级阅读模块手册。
3. 对每个模块，重点看：知识结构、关键机制、重要图示、易混淆点、简答题式总结。
4. 考前最后半天只看每个模块的“最后速记版”。

## 文件说明

| 文件 | 用途 |
|---|---|
| `00_课程总览.md` | 课程地图、模块关系、复习优先级 |
| `01_<模块>.md` | 模块知识手册 |
| `images/` | 课件图示或图片引用 |
| `source_map/` | 模块与课件来源映射 |

## 复习建议

- 不要一开始就陷入细节，先看总览。
- 每学一个模块，先看模块知识树，再读概念和机制。
- 对流程、算法、公式类内容，要能按步骤复述。
- 对图示类内容，要能解释图中每个组成部分或箭头关系。
- 对简答题模板，要练习用自己的话写出来。
